# SwiftShop

A modern, high-performance e-commerce platform for electronics and lifestyle products.

## Features

- **Modern UI**: Clean, responsive design built with React and Tailwind CSS.
- **Product Catalog**: Dynamic product listings with categories and details.
- **Shopping Cart**: Fully functional cart with quantity management.
- **Checkout Flow**: Streamlined checkout process.
- **Search**: Real-time product search.
- **Animations**: Smooth transitions powered by Framer Motion.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion, Lucide React.
- **Backend**: Express, SQLite (Better-SQLite3).
- **Build Tool**: Vite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
